## 프로젝트 소개

<h4>🛒shopshop </h4>

> 쇼핑몰 상품 주문하기 및 쇼핑몰 운영 플랫폼

1. 원하는 상품을 주문할 수 있습니다.
2. 관리자 모드에서 상품을 쉽고 빠르게 등록하고 삭제할 수 있습니다.
